# Phylogenetic Cards

Generate printable phylogenetic flash cards from tree data. Each card focuses on a clade (not individual species), with data sourced from Dawkins' *The Ancestor's Tale*.

Cards target Ivory Graphics Large Tarot size (89×127mm, 1051×1500px at 300 DPI) on 310gsm black-cored card stock.

## Usage

```bash
pip install -e .
phylo-cards --selector rendezvous --output output/
```
